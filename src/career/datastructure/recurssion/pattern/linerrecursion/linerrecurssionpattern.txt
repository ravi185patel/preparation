State reduces linearly (i, n, target)
Choices collapse using min / max / count

🎯 INTERVIEW CHECKLIST (MEMORIZE)

If a problem:
✔ Returns number / min / max
✔ State reduces linearly
✔ Choices collapse immediately
👉 Linear Recursion / Decision DP

If a problem:
❌ Returns lists
❌ Explores all paths fully
👉 Binary Recursion / Backtracking
